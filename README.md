# unfold-button-bar
可展開的button bar

源文件：UnfoldableButton.swift
包含 reuseable class UnfoldButton 

整頁複製後，在viewController 配置如下即可正常使用

var button:UIView = UnfoldButton(origin: CGPoint) //origin : buttonBar 在 superView 的位子。
